Trabalhando com listas

listas ordenadas

Listas ordenadas são listas que servem para organizar iténs que possue, sequencial lógico. A lista possue a TAG OL e os iténs tem a TAG

  1. primeiro itém
  2. segundo itém
  3. terceiro itém
  4. quarto itém

listas não ordenadas

Listas não ordenadas são listas que possuem uma sequência lógica, geralmente utilizam marcadores e usa a TAG UL para lista e LI para o itém

  • primeiro itém
  • segundo itém
  • terceiro itém
  • quarto itém
    • *
